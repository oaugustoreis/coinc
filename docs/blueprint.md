# **App Name**: Monthly Finances

## Core Features:

- Data Entry Form: Form to input expense details like month, paid status, account, card, value, and installment.
- Monthly Data Display: Display of financial data categorized by month.
- Month Selection: User can select the month from the app's UI
- Firestore integration: Integrate Firestore to persist the monthly transactions. Each transaction will be linked to the current user UID.
- Totals Calculation: Dynamically calculate and display total expenses, incomes and balance per month.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to represent trust and clarity in financial management.
- Background color: Very light gray (#F5F5F5) to ensure readability and reduce visual fatigue.
- Accent color: Green (#90EE90) for positive confirmations like successful data entry.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary feel. Use 'Inter' for body.
- Data entry form on the main page and monthly summaries on a separate tab or page.
- Use clear sections and tables to organize data, improving data presentation and overall UX
- Smooth transitions and loading animations to improve user experience.
